# qr-code-generator

Small API for QR codes generation. Developed using [FastAPI](https://fastapi.tiangolo.com/).

## Run with Docker example

    docker build -t qr-generator .
    docker run -d --name qrgenerator -p 80:80 -e PORT=80 qr-generator
